# Analysis-and-modeling-of-analog-modulators-and-random-signals
# Анализ и моделирование аналоговых модуляторов и случайных сигналов

**Тема:** Исследование нелинейных искажений, спектральных характеристик и статистических методов обработки сигналов  
**Методы:** полиномиальная аппроксимация, анализ Фурье, теория случайных процессов, байесовское обнаружение  
**Статус:** завершён

---

## Обзор

Проект состоит из четырёх независимых задач, охватывающих ключевые разделы радиотехники:

1. **Амплитудный модулятор** – аппроксимация ВАХ транзистора, расчёт статической модуляционной характеристики (СМХ), нелинейных искажений, спектра модулирующего сообщения и АМ‑сигнала.
2. **Частотный модулятор** – использование варикапа для генерации ЧМ, расчёт девиации частоты, спектра ЧМ‑сигнала (функции Бесселя).
3. **Случайное сообщение** – корреляционная функция, спектральная плотность мощности, дискретизация по Котельникову.
4. **Обнаружение сигнала** – байесовский порог, вероятности ложной тревоги и пропуска, минимизация ошибки.

Все расчёты приведены с промежуточными выкладками и графиками.

---

## Основные результаты

| Задача | Ключевой результат |
|--------|---------------------|
| АМ‑модулятор | Коэффициент нелинейных искажений K_ни = 5% при U_нч = 0.89 В, ширина спектра модулирующего сигнала Δf_эфф = 3 кГц |
| ЧМ‑модулятор | Индекс модуляции β = 14.5, практическая ширина спектра 2.2 МГц, доля энергии в боковых полосах ≈ 98% |
| Случайный сигнал | Интервал корреляции τ_к = 0.0345 с, энергетическая ширина спектра Δf_э = 29 Гц, частота дискретизации 58 Гц |
| Обнаружение | Оптимальный порог U_пор = 1.07 В, минимальная вероятность ошибки P_ош_min = 0.31 |
