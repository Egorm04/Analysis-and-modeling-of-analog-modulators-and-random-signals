# Частотный модулятор на варикапе

## Исходные данные

Тип варикапа: Д-902.  
Статическая вольт-фарадная характеристика (ВФХ) задана полиномом 3-й степени:

C(и) = c₀ + c₁u + c₂u² + c₃u³, где u – отрицательное напряжение, C – в пФ.

Коэффициенты:  
c₀ = 57.6 пФ  
c₁ = –4.5 пФ/В  
c₂ = 0.43 пФ/В²  
c₃ = –0.018 пФ/В³  

Несущая частота f₀ = 39 МГц.  
Параметры модулирующего сообщения – те же, что в задаче 1 (τ = 0.1 мс, T = 1 мс).  
Девиация частоты Δf_д = 144 кГц.

## 2. Выбор рабочей точки и расчёт индуктивности

Рабочая точка выбрана на середине прямолинейного участка СМХ:  
**U_см0 = –3.0 В**.

Статическая ёмкость в рабочей точке: C_стат = C(–3.0) = 57.6 –4.5·(–3) + 0.43·9 – 0.018·(–27) = 57.6 +13.5 + 3.87 + 0.486 = 75.456 пФ.

Индуктивность контура:  
L = 1 / ((2πf₀)²·C_стат) = 1 / ((2π·39·10⁶)²·75.456·10⁻¹²) ≈ 220.6 нГн.

После подстановки L скорректированная СМХ: f_рез(u) = 1 / (2π√(L·C(u))).

*Рис.13 – Скорректированная СМХ*

## 4. Спектр ЧМ‑сигнала при гармонической модуляции

Амплитуда несущей U₀ = 1 В, частота модуляции F = 1/T = 1 кГц (для удобства спектрального анализа используется гармоническое сообщение с амплитудой, дающей ту же девиацию).

Индекс модуляции: β = Δf_д / F = 144 / 1 = 144 (нормальный индекс для широкополосной ЧМ).  
Однако в задании дана девиация 144 кГц при F = 1 кГц, но в исходных данных (табл.2) частота модуляции для ЧМ не указана явно, поэтому расчёт выполнен для β = 144.  

Спектр ЧМ: U(t) = U₀ Σ_{k=-∞}^{∞} J_k(β) cos((ω₀ + kΩ)t).  
Функции Бесселя J_k(β) при β = 144 практически все равны 0 для |k| < 140, а затем появляются. Практическая ширина спектра по Карсону: Δf_ЧМ ≈ 2(Δf_д + F) = 2·(144+1) = 290 кГц.

*Рис.16 – Спектр ЧМ‑сигнала (огибающая амплитуд)*

Мощность в боковых полосах:  
P_бок = U₀²/2·(1 – J₀²(β)) ≈ 0.98·P_полн при β = 144. Доля энергии ≈ 98%.
