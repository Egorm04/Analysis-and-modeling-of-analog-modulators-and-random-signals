## 1. Аппроксимация ВАХ

Интервал входных напряжений [0; 2.8] В.  
Аппроксимация полиномом 4-й степени:  
I(u) = a₀ + a₁u + a₂u² + a₃u³ + a₄u⁴.

При u = 0, I = 0 ⇒ a₀ = 0.  
Система уравнений для остальных точек решена методом наименьших квадратов. Получены коэффициенты:

a₁ = 0.0732  
a₂ = 2.114  
a₃ = –1.894  
a₄ = 0.572

## 2. Статическая модуляционная характеристика (СМХ)

Амплитуда первой гармоники тока при гармоническом входном сигнале u(t) = U_см + U_m cos(ωt):

I₁(U_см) = (1/π) ∫₀^{2π} I(U_см + U_m cosθ) cosθ dθ.

Подставляя полином и интегрируя, получаем:

## 3. Выбор рабочей точки и нелинейные искажения

Рабочая точка выбрана в середине прямолинейного участка СМХ: **U_см0 = 1.2 В**.

При гармоническом модулирующем сигнале u_Ω(t) = U_Ω cos(Ωt) огибающая I₁(t) = I₁(U_см0 + U_Ω cosΩt).  
Разложение в ряд Фурье по частотам kΩ даёт коэффициент нелинейных искажений:

K_ни = √(Σ_{k=2}∞ I_{1k}²) / I_{11}.

Зависимости K_ни(U_Ω) и глубины модуляции M₁(U_Ω) рассчитаны для 8 значений U_Ω, выходящих за пределы линейного участка.

## 4. Спектр модулирующего сообщения

Сообщение – прямоугольные импульсы (A = 1, τ = 0.1 мс, T = 1 мс).

Спектр (ряд Фурье):  
c₀ = τ/T = 0.1  
c_k = (2Aτ/T)·|sin(πkτ/T)/(πkτ/T)| = 0.2·|sinc(0.1πk)|.

Энергия на интервале (-τ/2, τ/2): E_х = A²τ = 0.1 мДж.  
Энергия в спектре без постоянной составляющей: E_перем = 0.09 мДж.

Эффективная ширина спектра по критерию 90% энергии – до 9-й гармоники (k = 9), т.е. **Δf_эфф = 9 / T = 9 кГц**.

Добротность контура модулятора:  
Q = f₀ / (2 Δf_эфф) = 39·10⁶ / (2·9·10³) ≈ 2167.

## 5. Амплитудный спектр и временная диаграмма АМ‑сигнала

Амплитуда несущей U_нес = 0.89 В (выбрана из условия K_ни = 5%).  
Эквивалентное сопротивление контура R_экв = 1000 Ом.

Временная диаграмма:  
u_АМ(t) = U_нес·[1 + M·x(t)]·cos(2πf₀t), где M = I₁(U_Ω)/I₁(U_см0) – глубина модуляции.
